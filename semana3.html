Semana 3

Laboratorio de innovación 2

Jueves 14 de Septiembre

Vimos ejemplos de proyectos tecnológicos que hicieron otros grupos en ciclos pasados que estaban realizados con Arduino para resolver problemas que cubran alguna ODS.

También aprendimos la diferencia entre JavaScript, que es para la parte Funcional, CSS, que es para la estética y el HTML, que sería la estructura de nuestra web. Tocamos sobre todo, temas de HTML para poder programar nuestra bitácora virtual a base de una plantilla, y también un poco de CSS para darle una presentación más atractiva y acorde a nuestro estilo..

Repasamos componentes de Arduino que vimos en Laboratorio de Innovación 1 y vimos varios nuevos para saber su funcionalidad y en qué tipo de casos son usados unos u otros. Como por ejemplo la diferencia entre el sensor de movimiento y de presencia, donde el de movimiento sería el más óptimo para las luces de un pasillo, que se vayan prendiendo y apagando automáticamente según vayas pasando, y el sensor de presencia sería para una habitación donde te vas a quedar y quieres que las luces se mantengan encendidas mientras estés allí.

Además vimos nuevas maneras de usar prompts de Chat GPT como el de que asuma una personalidad o especialidad profesional para que brinde respuestas distintas y más de acorde a lo que buscamos..

Sábado 16 de Septiembre

Hicimos un ejercicio práctico para cambiar los textos, imágenes y funciones de botón cambiando el código de la plantilla elegida en Visual Studio Code para personalizar nuestra bitácora.
